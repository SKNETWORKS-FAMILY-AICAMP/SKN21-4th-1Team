# 시연 자동화 가이드

사용자가 챗봇과 상호작용하는 것처럼 보이도록 시뮬레이션하는 **고스트 커서 자동화 스크립트**를 구현했습니다. 이 스크립트는 브라우저 내부에서 작동하므로 외부 녹화 도구의 부자연스러운 흔적(파란 테두리 등)이 남지 않습니다.

## 데모 실행 방법

1.  **Django 서버 실행** (이미 실행 중이라면 생략):
    ```bash
    python manage.py runserver
    ```
2.  **`demo=true` 파라미터와 함께 채팅 페이지 열기**:
    -   URL: `http://127.0.0.1:8000/chat/?demo=true`
    -   (또는 `http://localhost:8000/chat/?demo=true`)

3.  **화면 녹화 준비**:
    -   페이지가 로드되고 약 1초 후 **고스트 커서(가짜 마우스)**가 나타납니다.
    -   자동으로 "새로운 채팅"을 클릭합니다 (페이지 새로고침).
    -   3가지 질문을 차례로 입력하고 전송합니다.
    -   마지막으로 **휴지통 아이콘(삭제)**을 클릭하고 **로그아웃**합니다.

## 구현된 기능

-   **고스트 커서 (Ghost Cursor)**: 실제 마우스 커서 대신 가짜 커서가 움직입니다.
-   **포커스 링 제거**: 입력창 클릭 시 발생하는 파란색 테두리(outline)를 CSS로 숨겼습니다.
-   **시나리오 순서**:
    1.  새로운 채팅 클릭
    2.  질문 1: "근로계약서 미작성 시 벌금은 얼마인가요?"
    3.  질문 2: "퇴직금 지급 기준이 어떻게 되나요?"
    4.  질문 3: "주휴수당 계산법 알려줘"
    5.  세션 삭제 (UI 시뮬레이션)
    6.  로그아웃
-   **삭제 버튼**: 사이드바 세션 목록에 휴지통 아이콘을 추가했습니다. (데모용으로 클릭 시 화면에서만 사라지도록 처리됨)

> [!NOTE]
> 타이핑 속도와 마우스 움직임은 리얼함을 위해 약간의 무작위성이 적용되어 있습니다.
